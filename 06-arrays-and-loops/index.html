Arrays and Loops

Creation using array literal

const array = ["apple", "orange", 10, 200n, true, false, null, undefined]

console.log(array)

const array1 = ["apple", 200, true, array, function() { return 10; }]

console.log(array1)

const array2 = new Array("apple", 200, true, array, function() { return 10; }, null, undefined)

console.log(array2)

console.log(array1)

Looping over items in an array

Using for loop and length property:

console.log("--------------------")

for (let i = 0; i < array.length; i++) {

console.log(array[i]);

}

console.log("--------------------")

Using the forEach() method:

array2.forEach(item => console.log(item))

console.log("--------------------")
